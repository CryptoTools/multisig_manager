Select Crypto allows you to choose which crypto currency you want to work with. 

Tasks are peformed for the crypto currency that you select. Below are the available tasks:

Create Public/Private Key Pair, create a public/private key pair to be used in a multisignature address. 
The key pair will be stored on disk. Private key can be chosen to be encrypted with a password. 

List Public/Private Key Pair. List all the public/private key pair that have been created. 

Create Multisignature address. Create a multisignature address. 

List Multisignature Address. List all the multisignature addresses that have been created 

Send transaction from multisignature address. Send a transaction from a multisignature address
